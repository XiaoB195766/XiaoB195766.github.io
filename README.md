# AI Agent 系统终极演示 v3

这是一个完整的 **AI Agent 系统演示页面**，模拟真实 SaaS 产品后台，可用于展示自动化任务、Agent 运行、Token 统计、虚拟作业提交等功能。

## 功能亮点

- 登录页 → 仪表盘  
- 实时任务进度条和 Agent 控制按钮  
- 虚拟操作：提交作业 / 生成报告  
- Token 消耗动态增长  
- 系统日志自动滚动  
- 用户列表及状态展示  
- 数据可视化图表（柱状图 + 饼图）  
- 聊天窗口，可模拟与 AI Agent 对话
